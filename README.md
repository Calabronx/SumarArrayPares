# SumarArrayPares
Un ejercicio programado en java, para sumar una estructura de un array en pares.

Array dado : { 2, 2, 5, 4, }

Resultado:
{ 2, 2, 5, 4, }


{ 2, 2 } --> 4


{ 5, 4 } --> 9

Realizado de una pagina llamada Edabit
